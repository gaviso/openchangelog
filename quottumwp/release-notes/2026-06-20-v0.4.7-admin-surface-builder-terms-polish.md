Cross-workspace operator console + everything that landed since v0.4.6 (WYSIWYG terms, quote builder rebuild, click-to-edit line items, units).

### Admin surface (new — /admin)

A separate operator-mode section, visible only to **platform_admins**. Red accent strip on the topbar + \"Operator mode\" chip so you always know you're not in a customer workspace.

- **/admin** — platform overview: workspaces, members, MRR, active contracts.
- **/admin/workspaces** — list every workspace platform-wide with status filter tabs (Active / Trial / Suspended / Archived), KPI strip, sortable register, text search.
- **/admin/workspaces/[id]** — workspace detail: header chips, KPIs, members, settings inspector, recent admin_audit. **Action buttons**: Impersonate · Suspend · Restore · Archive — each logs to admin_audit.
- **/admin/users** — every signup with last sign-in, email-confirmed state, and the workspaces they belong to.
- **/admin/audit** — recent operator actions.

**Impersonation** sets the workspace cookie + a marker cookie (4hr cap). The customer dashboard then renders a red banner \"Operator: viewing as {workspace name}\" on every page, with a one-click **Stop** button. Both start and stop log to admin_audit.

Service-role client (server-only) bypasses RLS for /admin/* queries. platform_admins table is itself service-role-only — no client-side leakage of admin membership.

### Polish landed since v0.4.6

- **WYSIWYG editor for Terms & Conditions** (TipTap) — landed on Settings → Terms; PDF renders the markup.
- **Quote builder rebuild** — two-column layout with sticky **Live Preview** sidebar, library search bar for line items, click-to-edit rows with inline TipTap description editor, **Unit picker** (per project / per year / per month / per hour / each / custom).
- **Quote detail page rebuild** — full layout matching the design: header + KPI strip + line items / totals / payment plan / attachments stub / activity sidebar + Term & Billing card + Next-Step card with Copy/Rotate.
- **Expandable line-item rows** on the quote detail page — click name to reveal Markdown description.
- **Drag-to-reorder** + **inline edit** on the Categories settings page.
- **Coupons**: click-to-copy code + Generate-random-code button.
- **WYSIWYG-friendly Markdown rendering** via a shared <RichTextRender> that sniffs HTML vs Markdown and routes accordingly.
- **unit_label** added to both line_items library + quote_items (migrations 011 + 012). Library backfilled with sensible defaults. Library picks now carry units through to quotes.
- Settings nav cleanup: renamed \"Phases\" → \"Categories\" and \"Service Group library\" → \"Service Groups\". Removed per-workspace **Email (SMTP)** and **Daily sweep** tabs — moving to admin settings.
