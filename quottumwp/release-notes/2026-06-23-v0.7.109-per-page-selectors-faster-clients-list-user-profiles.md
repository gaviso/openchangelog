## What's new
- **Rows per page on every list** — Clients, Quotes, Contracts (and the operator console lists) now have a **25 / 50 / 100** selector, defaulting to 25.
- **Faster Clients list** — it now loads only the page you're viewing, with server-side search and sorting, so large client lists stay snappy.

## Fixes & polish
- Dates now render in a **consistent US format** (e.g. "Jul 15, 2026") everywhere — in the app, on the public quote pages, and in emails.

## Operator console
- A new **user profile page** — open any user from the Users list to see their workspaces, roles, beta access, and recent activity in one place.
