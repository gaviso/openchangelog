### Added
- **Client terms acceptance** — required checkbox on the quote page that gates **Accept** (client + server enforced, logged to activity); display-only checkbox on the contract page.
- **Footer-note on/off toggle** — the footer note now renders independently of the master Terms toggle, on quotes and contracts (page + PDF).
- Quote **"Prepared for"** block shows company location, website, phone and the primary contact.
- Terms & Conditions settings now also drive the **contract document** (page + PDF).

### Changed
- Client quote items reordered to **Unit price → Qty → Line total**.
- New quotes default the **Service period to "One time"**.
- Quote detail admin page widened to **1440px**.

### Fixed
- Admin notices no longer overlap the page layout (relocated into the hero heading).
- The line-items **Amount column no longer clips** on long item names.

See `CHANGELOG.md` for full detail.
