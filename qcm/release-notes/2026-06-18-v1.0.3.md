### Changed
- **Send quote** modal — the message body is now a formatted rich-text editor (WordPress TinyMCE): sans-serif content font, a simple toolbar (bold / italic / lists / link), and no Code tab. The edited HTML is still wrapped in the master email shell on send.

### Fixed
- **Dashboard** — long quote/contract titles now truncate with an ellipsis (full title on hover) instead of widening the table and pushing the Status column out of the card.
