### Added
- **Client email templates** — a new *Email templates* settings tab with six editable templates (Quote sent / reminder / accepted / declined / revision requested, and Contract renewal). Each has an editable subject + HTML body, a **live preview**, and a **Send test** button to any address.
- **Master HTML email shell** with a `{{message}}` slot, edited in a code editor (CodeMirror) with a live preview — every client email is wrapped in it and sent as HTML.
- **Automatic quote reminders** (configurable cadence, default 3 & 7 days) on the renamed **Reminders** tab, plus client-facing **contract renewal reminder** emails at each milestone.
- Per-template **auto-send** on/off toggle and an optional **require-terms-acceptance** setting.
- **"New version available"** updater notice in the sidebar.

### Changed
- The Send-quote email and all automatic client emails now send as HTML, wrapped in the master shell.
- The *Renewals* settings tab is renamed **Reminders**.

See \`CHANGELOG.md\` for full detail.
