Service-taxonomy import and contract-renewal polish.

### Added
- **Pull from Baserow** (Settings → Data & tables) — a unified card to import the service taxonomy: select any of Categories / Service groups / Line items (or all), pulled in dependency order so parent links resolve. Two modes: **Add & update** (create missing, update matches, keep the rest) and **Replace** (mirror Baserow — also remove rows in the selected tables that aren't in the pull, with a confirmation step). Results report created / updated / removed.
- **Contract renewal banner** — a renewed contract now shows "This contract was renewed into <successor> →" and the successor shows "This contract is a renewal of <predecessor> →", each linking through. Numbers are computed live, so they're always correct.

### Changed
- **Auto-renew** is disabled once a contract has been renewed (it's superseded by its successor) — enforced in the UI and the toggle handler.
- **Data & tables layout** — the Baserow connection/pull and the Export/Restore cards are paired as equal-height two-column rows.
